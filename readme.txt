Display relevant courses and trainings from Springest on your website.

== Description ==

This plugin lets you create an automatically generated set of pages on your WordPress website with a listing of relevant trainings and courses from Springest. Our powerful API provides you with extreme flexibility in the selection of the trainings and courses to be displayed.

To use this plugin you will need a Springest API key. If you do not have one yet, please contact our Support Team at support@springest.nl to request one.


= Features =

* Easily display relevant courses from Springest on your website
* Create your own custom groups/pages for the trainings you want to display
* Use advanced search queries to select extremely relevant trainings and courses from the Springest API
* Choose between our English, German, Dutch and Belgian API's
* Seamlessly integrated with WordPress


== Installation ==

1. Download the plugin and unzip
1. Place the springest-partners folder in your wp-content/plugins folder
1. Activate the plugin through the WordPress admin
1. Create a standard Page which you would like to use as the base of your course index, for instance www.example.com/trainings
1. Go to Settings > Springest in your WordPress admin to choose your API and enter the API key you received from Springest and select the base page you created in the previous step
1. Go to Springest in the left menu of your WordPress admin to create custom groups and choose the trainings you wish to be displayed on your website
1. All set! Go to your website and check it out!

If you have any questions or remarks, please feel free to contact us at any time support@springest.nl
